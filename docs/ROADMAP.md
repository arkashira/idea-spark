# ROADMAP.md

## Roadmap

### MVP Milestone (Must-Have for Launch)

The Minimum Viable Product (MVP) will focus on delivering a functional idea tracker that integrates with Axentx OS product pipeline. The following features are critical for the MVP:

#### 1. Idea Tracking

* Users can submit and track ideas
* Ideas are linked to Axentx OS product pipeline
* Basic idea metadata (title, description, tags) is stored

#### 2. 90-Day Conversion Rate Tracking

* Automated tracking of 90-day conversion rates for submitted ideas
* Conversion rates are displayed on the idea dashboard

#### 3. NSM Report Generation

* Basic NSM reports are generated for performance analysis
* Reports include conversion rates, idea submission trends, and other key metrics

### v1 Phase: Enhancements and Refinements

The v1 phase will focus on refining the MVP and adding new features to enhance the user experience.

#### 1. Idea Prioritization

* Users can prioritize ideas based on relevance, feasibility, and business value
* Prioritized ideas are displayed on the dashboard

#### 2. Customizable Dashboards

* Users can create custom dashboards to visualize key metrics and trends
* Dashboards can be shared with team members or stakeholders

#### 3. Integration with Axentx OS Brain

* Integration with Axentx OS brain for real-time idea validation and feedback
* Brain insights are displayed on the idea dashboard

### v2 Phase: Advanced Features and Scalability

The v2 phase will focus on adding advanced features and improving scalability to support growing user bases.

#### 1. Machine Learning-based Idea Filtering

* Machine learning algorithms filter and prioritize ideas based on user behavior and Axentx OS brain insights
* Filtered ideas are displayed on the dashboard

#### 2. Collaborative Idea Development

* Users can collaborate on idea development and track progress
* Collaboration features include commenting, @mentions, and task assignment

#### 3. Advanced NSM Reporting

* Advanced NSM reports include detailed analytics and insights on idea performance
* Reports can be customized to meet specific business needs

### Theme: Continuous Improvement

Throughout all phases, we will focus on continuous improvement, gathering user feedback, and iterating on the product to ensure it meets the evolving needs of our users.

### Theme: Integration with Axentx OS

We will continue to integrate Idea Tracker with Axentx OS, leveraging its capabilities to enhance the idea tracking and validation process.

### Theme: Scalability and Performance

We will prioritize scalability and performance to ensure the product can handle growing user bases and increasing data volumes.
