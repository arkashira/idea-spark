# breakeven.md

## Unit Economics & Break-even Analysis for Idea-Spark

### Cost per Active User
- **Compute Costs**: $0.10/user/month (based on estimated cloud compute usage for AI processing)
- **Storage Costs**: $0.05/user/month (for user data and generated ideas)
- **Bandwidth Costs**: $0.02/user/month (for data transfer and API calls)

**Total Cost per Active User**:  
**$0.10 + $0.05 + $0.02 = $0.17/user/month**

### Pricing Tiers
| Tier Name        | Price ($/mo) | Features                                           |
|------------------|---------------|----------------------------------------------------|
| Basic            | $10           | Idea generation, Basic validation tools            |
| Pro              | $25           | All Basic features + Advanced validation tools, Collaboration features |
| Premium          | $50           | All Pro features + Personalized coaching, Priority support |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: $30 - $50 per user (includes marketing, sales efforts, and onboarding costs)

### Lifetime Value (LTV) Estimate
- Average user lifespan: 24 months
- Average revenue per user (ARPU): $25 (assuming a mix of tiers)
  
**LTV Calculation**:  
**LTV = ARPU × Average Lifespan = $25 × 24 = $600**

### Break-even Users Count
**Break-even Point Calculation**:  
- Fixed Costs (monthly operational costs): $1,000 (including marketing, salaries, etc.)
- Contribution Margin per User:  
  - Average Revenue per User (ARPU): $25  
  - Total Cost per Active User: $0.17  
  - Contribution Margin = ARPU - Total Cost = $25 - $0.17 = $24.83

**Break-even Users Count**:  
**Break-even Users = Fixed Costs / Contribution Margin = $1,000 / $24.83 ≈ 40 users**

### Path to $10K MRR
To achieve $10,000 in Monthly Recurring Revenue (MRR):

1. **Basic Tier**:  
   - Price: $10  
   - Users Needed: $10,000 / $10 = 1,000 users

2. **Pro Tier**:  
   - Price: $25  
   - Users Needed: $10,000 / $25 = 400 users

3. **Premium Tier**:  
   - Price: $50  
   - Users Needed: $10,000 / $50 = 200 users

**Recommended Path**:  
Focus on the **Pro Tier** with **400 users** for a balanced approach that includes advanced features appealing to indie hackers and solo creators.
