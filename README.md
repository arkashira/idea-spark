# Idea Spark
Idea Spark is a core idea generation engine that generates three AI-generated product ideas within 30 seconds based on a brief description of interests.

## Usage
To use Idea Spark, simply run the `main.py` file with the `python` command. This will generate three ideas based on the default interests.

## Testing
To run the tests, use the `pytest` command. This will run all the tests in the `tests` directory.
